# A. Toggling items.

```
 function handleToggleItem(id) {
    setItems((items) =>
      items.map((item) =>
        item.id === id ? { ...item, packed: !item.packed } : item
      )
    );
  }
```

# B. Delete items

```
 function handleDeleteItem(id) {
    setItems((items) => items.filter((item) => item.id !== id));
  }
```

# C. Update items

```
 function handleAddItems(item) {
    setItems((items) => [...items, item]);
  }

```
